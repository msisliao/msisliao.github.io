flex盒子模型解析

flex-flow 布局方向

flex-direction 方向：水平或者垂直
flex-wrap ：是否换行 wrap nowarp

//示例：
ul{
  flex-flow: row  wrap;
}
//相当于
ul{
   flex-direction: row;
   flex-wrap: wrap;
}
